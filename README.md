### Kitty Configuration

Navigation between panes/nvim:
```
Ctrl + h/j/k/l
```
Navigation between tabs:
```
Ctrl + Shift + left/right
Ctrl + TAB
```

New tab:
```
Ctrl + Shift + t
```

Alternate layout(zoom):
```
Ctrl + Shift + l
```
